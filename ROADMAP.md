# Roadmap

## Version 1.0 (Current)

**Foundation & Core Providers**

- [x] Telegram bot integration
- [x] RBAC auth system
- [x] Connection management
- [x] Credential encryption
- [x] Rate limiting
- [x] Audit logging
- [x] Cloudflare provider (DNS, Workers, R2, D1, KV, Pages)
- [x] Firebase provider (Firestore, Storage, Auth)
- [x] GitHub provider (Repos, Orgs, Search)
- [x] CI/CD pipelines
- [x] Documentation

## Version 1.1

**Provider Expansion**

- [ ] Cloudflare Cache management
- [ ] Cloudflare Analytics
- [ ] Cloudflare Stream

- [ ] Firebase Cloud Functions management

## Version 1.2

**Advanced Features**

- [ ] D1 query builder UI in Telegram
- [ ] Cloudflare Workers editor (create/edit/deploy via chat)

- [ ] Web dashboard (minimal read-only)

- [ ] Connection templates

**Enterprise Features**
